> `win+R`快捷键可以打开运行窗口，运行窗口下支持输入命令，快捷打开很多程序

1. 系统自带程序
2. 设置在环境变量下的用户程序

   2. 配置在环境变量下的文件

### **3、Excel下的函数**

函数都是以`=`号开头，然后输入行数名 ，比如：求和函数

```
// 计算第A列，第一行到第九行的和
=sum(A1:A9)   // 写完后回车

```
